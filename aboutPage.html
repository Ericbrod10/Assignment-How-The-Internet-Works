About the Project

The initial project was to give a rudimentary view on the history of the internet and its impacts on the world. This is seen in the invention of the internet, browser wars, internet search and people power sections. I took the initial site we had created and cloned it for the second assignment where I essentially merged both websites together to give a better description and overview of the internet. The main focus around the second assignment entails Domain names and ICANN, IP Addresses, Packet, Routing, DNS, HTTP/S, HTML and CSS. Note: the professor mentioned in a previous lecture we could take our previous project and merge it with our current assignment so I did that to get more practice in integrating new features into existing websites.

The first topic is about the Domain names and ICANN. Here we will take a look at how ICANN is the Internet Corporation for Assigned Names and Numbers maintains the central repository for IP addresses which is essential to the internet.

The next topic is IP Addresses, Packets and Routing. This section looks into what are IP Addresses and how Packets and Routers use them to transport data through the internet.

After getting into what IP Addresses are, we take a look at what a DNS server is and how it is almost essential for browsing the web.

Once we discuss how DNS servers help the web, we start looking into the foundation of how data is being transferred over it with HTTP.

Lastly, we discuss what W3C is, and how HTML and CSS make up the basic structure of almost all webpages on the internet.

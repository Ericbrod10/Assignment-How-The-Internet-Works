HTTP and HTTPS Protocols

What is HTTP? HTTP is also known as Hyper Text Transfer Protocols, and can be thought as the language to communicate between web browsers and servers. The HTTP messages sent between a web browser and server are essentially get requests which tells a server to send the browser the specific html code that the user wants to see. Some may have seen HTTPS websites; this is essentially HTTP code but it is secured and runs on port 443 as opposed to port 80 for normal HTTP.

IP Addresses, Packets and Routing

What are IP Addresses and what is their purpose? An IP Address is an Internet Protocol address which is a numerical assigned to each device connected to a network that uses the internet. An IP address serves two main functions: host or network interface identification and location addressing. When downloading data or information from a website, the IP address helps the website know where to actually send that information otherwise it would be impossible to send anything.

When sending a large file on the internet, the file could be broken up into a number of packets depending on how large the file is. A packet is a piece of data that travels on the internet and it has the IP address of where it came from and the IP address of where it is being sent. Special computers on the internet called routers help direct each packet to reach their destination in the quickest route at the time. The route that each packet takes could be the same or a completely different than the other packets being sent with it, since the internet is a constantly changing web of connections being established or dissolved, routers are a fundamental reason why the internet is so reliable.

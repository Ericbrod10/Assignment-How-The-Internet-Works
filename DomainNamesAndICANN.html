Domain names and ICANN

What is ICANN? ICANN is the Internet Corporation for Assigned Names and Numbers. They are a nonprofit organization in southern California that manages the Internet’s core infrastructure. ICANN maintains the central repository for IP addresses and helps coordinate the supply of IP address all over the world. They all manage the domain name system (DNS) and root servers of the internet.

What are Domain Names? A domain names is an address that users can use to find a website. On the internet, computers use IP addresses which are a series of numbers to locate other websites. This is however very difficult for people to do since it involves memorizing strings of numbers. This is how domain names came into user. A domain name essentially is a name such as “Google” which is associated with Google’s IP, so someone who wants to visit Google’s site would just need to remember instead of the numerical string associated with it.
